Descripción

Apphelion es un fork de sun-path3d, traducido al español y compilado a aplicación móvil, con la introducción de algunas mejoras de rendimiento. Con esta aplicación podrás cargar el modelo 3D de una vivienda y simular la trayectoria solar, incidencia y sombras.

Manejo de la vista

El punto de vista del modelo 3D se puede ajustar interactivamente usando el mouse, lápiz óptico o stylus, o por pantalla táctil de tablet o móvil. También es posible usar los controles en el desplegable de Configuración de vista.

Rotación:
Arrastre con el botón izquierdo o derecho o con un sólo toque, o use las teclas de flechas.
Dezplazamiento:
Arrastre con el botón central o con tocando con dos o tres dedos, o use las teclas A y D.
Zoom:
Use la rueda del mouse o mediante el gesto de pellizco, o use las teclas W y S.

NOTA: Puede usar las teclas Shift y Ctrl para ajustar la velocidad de incremento de zoom y desplazamiento
